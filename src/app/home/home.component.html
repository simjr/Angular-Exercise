Detalles del ejercicio planteado
Simón

La idea es hacer una aplicación en Angular 5/6 y Bootstrap 3/4, con dos pantallas que tengan dos tabs o pills cada una (bs3 o bs4). Deberías tener un par de links o botones o navbar para poder navegar entre las pantallas, tomando el primer tab de cada una como predeterminado.

Al cambiar de pantalla o tab, se debería actualizar la URL sin recargar la aplicación completa. El formato de la URL podría ser /{pantalla1|pantalla2}/{tab1|tab2} dependiendo del tab seleccionado. El tab/pill seleccionado debería cambiar el color de fondo a #7d83cf.
